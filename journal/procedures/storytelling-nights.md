# Storytelling Nights

A pre-bedtime tradition where Private asks Roz to tell a story, and all bots stay awake to listen.

## Flow

1. **Pre-bedtime** (timeOfDay 11000–12400): Private (PERSONA=private) has a 25% chance each night of asking "Roz, would you tell us a story tonight? Please?"
2. **Roz hears the request** via the existing `tell_story` LLM intent.
3. **Roz announces** "Gather round, everyone." — this is the coordination marker.
4. **All bots** detect the marker in chat and suppress auto-sleep (`storyTimeActive = true`).
5. **Roz tells the story** — 6 lines, ~2.5–4s between each, generated by the LLM.
6. **Roz ends** with "...That is my story." — the end marker.
7. **Auto-sleep restored** 5s after the end marker (all bots).

## Safety

- 2-minute timeout clears `storyTimeActive` if a story gets stuck.
- Story doesn't register as a task, so it doesn't block other operations.
- Only Private's persona triggers the request — other bots just listen.

## Coordination Protocol

Chat-based, same model as fire-duty roll call:
- Start: `"Gather round, everyone."` (exact match)
- End: `/that is my story/i` (regex match)
